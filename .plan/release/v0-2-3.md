# version 0.2.3

## features

- **Command reference documentation**: Complete list of available commands
- **Quickstart guide**: Getting started instructions for new users
- **Helpful no-command message**: Shows plugin name, version, and usage hint when no command specified
- **Version history**: Detailed changelog in plan
- **SDK integration**: Now respects `WAVE_DEBUG=1` for JSON error output (inherited from wave-core SDK)
- **Watch mode**: Commands with `watch` patterns auto-restart on file changes; use `--watch` flag to enable

## fixes

- **Clean error messages**: Simplified "command not found" error without debug codes
- **Watch process restart**: Fixed watch mode to properly kill and restart processes
- **Shared exec helpers**: Deduplicated shell command setup in flow runtime
